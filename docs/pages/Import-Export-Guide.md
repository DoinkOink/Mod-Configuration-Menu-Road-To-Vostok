## How To Import Mod Configs
1. Open MCM
2. Click on the "Import" button in the top right

![](https://github.com/DoinkOink/Mod-Configuration-Menu-Road-To-Vostok/blob/main/docs/assets/images/wiki/import-export-mcm.png?raw=true)

3. Navigate to the config file you've downloaded, select it, then press the "Open" button

![](https://github.com/DoinkOink/Mod-Configuration-Menu-Road-To-Vostok/blob/main/docs/assets/images/wiki/import-select-screen.png?raw=true)

4. Press the "Import Config" button and it will import all of the selected mods settings and take you back to MCM
    * (Optional) If you wish to not import specific mods config settings you can uncheck the checkmark next to its name

![](https://github.com/DoinkOink/Mod-Configuration-Menu-Road-To-Vostok/blob/main/docs/assets/images/wiki/import-confirmation-screen.png?raw=true)

## How To Export Mod Configs
1. Open MCM
2. Click on the "Export" button in the top right

![](https://github.com/DoinkOink/Mod-Configuration-Menu-Road-To-Vostok/blob/main/docs/assets/images/wiki/import-export-mcm.png?raw=true)

3. Press the "Export Config" button
    * (Optional) If you wish to not export specific mods config settings you can uncheck the checkmark next to its name
    * (Optional) If you want MCM to automatically open the folder you save the file to after saving click the "Open file directory on save" checkmake above the "Export Config" button

![](https://github.com/DoinkOink/Mod-Configuration-Menu-Road-To-Vostok/blob/main/docs/assets/images/wiki/export-screen.png?raw=true)

4. A file dialogue will open asking you to select the folder you wish to save the file at and the name you wish to save the file as
5. Click "Save" when ready to save and it will take you back to MCM while also opening the folder if previously selected

![](https://github.com/DoinkOink/Mod-Configuration-Menu-Road-To-Vostok/blob/main/docs/assets/images/wiki/export-save-screen.png?raw=true)
